OVERVIEW
What the tool does, what it's made of, and what you can customize
What It Does
The Full Flow
User clicks a button → UserForm opens with plan detail fields and an email picker. Fill in case #, plan name, TM, TC, optional amount → select the email type → click Send. The macro pulls the right template, swaps all tokens, generates a custom follow-up message, opens a pre-addressed Outlook draft, and creates a calendar appointment for follow-up — all in one shot.
What You Can Customize
Touch Points
  • Add / remove TM or TC — edit GetTMList / GetTCList functions
  • Add a new email type — add template file + entry in GetTemplateAndMessage
  • Add new tokens — update ReplaceTokens with new {token} key
  • Change follow-up timing — edit days in CreateFollowUpAppointment
  • Toggle success popups — comment out MsgBox calls in email builder
  • Add dynamic date tokens — use {date+N} pattern in any template

mod_TokenEngine.bas 
' ============================================================
' mod_TokenEngine
' ReplaceTokens() — takes the raw template string and swaps
' every {token} with its live value.
'
' Static tokens: {case} {plan} {TM} {TC} {amount}
' Dynamic tokens: {date+N} where N = days to add to today
'   Examples:
'     {date+0}  → today's date
'     {date+2}  → two business days from today
'     {date+10} → 10 days from today
'
' To add a new token:
'   Add a Replace() call below, or extend the dynamic parser
'   for more complex patterns
' ============================================================

Option Explicit

Public Function ReplaceTokens(ByVal body As String, _
                               caseNum As String, _
                               planName As String, _
                               tmName As String, _
                               tcName As String, _
                               amount As String) As String

    ' ── Static token replacement ──────────────────────────────
    body = Replace(body, "{case}",   caseNum)
    body = Replace(body, "{plan}",   planName)
    body = Replace(body, "{TM}",    tmName)
    body = Replace(body, "{TC}",    tcName)
    body = Replace(body, "{amount}", amount)

    ' ── Dynamic date token replacement ────────────────────────
    ' Scans for any {date+N} pattern and resolves it to a date string
    ' Example: {date+10} becomes "03/20/2026" if today is 03/10/2026
    Dim i As Long
    Dim tokenStart As Long, tokenEnd As Long
    Dim token As String, daysStr As String
    Dim daysToAdd As Long

    Do
        tokenStart = InStr(body, "{date+")
        If tokenStart = 0 Then Exit Do

        tokenEnd = InStr(tokenStart, body, "}")
        If tokenEnd = 0 Then Exit Do

        token   = Mid(body, tokenStart, tokenEnd - tokenStart + 1)
        daysStr = Mid(token, 7, Len(token) - 7)
        If Not IsNumeric(daysStr) Then Exit Do

        daysToAdd = CLng(daysStr)
        body = Replace(body, token, Format(Date + daysToAdd, "mm/dd/yyyy"))
    Loop

    ReplaceTokens = body

End Function

Email Builder & Appointment Creator
Main assembly sub, follow-up appointment creator, and form launcher entry point

mod_EmailBuilder.bas 
' ============================================================
' mod_EmailBuilder
' The orchestration layer. Three subs:
'   BuildAndSendEmail()         — main assembly + dispatch
'   CreateFollowUpAppointment() — SLA-based calendar entry
'   EmailGeneratorMain()        — form launcher (wire your button here)
' ============================================================

Option Explicit

' Template files folder — update this path to your templates location
Private Const TEMPLATE_PATH As String = "C:\Templates\EmailGen\"

' ----------------------------------------------------------
' BuildAndSendEmail
' Called by the UserForm submit button.
' ----------------------------------------------------------
Public Sub BuildAndSendEmail(ByVal emailType As String, _
                              caseNum As String, _
                              planName As String, _
                              tmName As String, tmEmail As String, _
                              tcName As String, tcEmail As String, _
                              amount As String)

    Dim templateFile As String, customMsg As String
    Dim body As String
    Dim mailItem As Outlook.MailItem

    ' 1. Look up template + custom message
    GetTemplateAndMessage emailType, templateFile, customMsg

    If templateFile = "" Then
        MsgBox "No template configured for: " & emailType, vbExclamation
        Exit Sub
    End If

    ' 2. Load template from disk
    body = ReadTextFile(TEMPLATE_PATH & templateFile)

    ' 3. Replace all tokens
    body = ReplaceTokens(body, caseNum, planName, tmName, tcName, amount)

    ' 4. Show custom message / reminder before sending
    If customMsg <> "" Then
        MsgBox customMsg, vbInformation, "Before You Send"
    End If

    ' 5. Build Outlook mail item
    Set mailItem = Application.CreateItem(olMailItem)
    With mailItem
        .To = tmEmail & ";" & tcEmail
        .Subject = emailType & " — Case #" & caseNum & " — " & planName
        .Body = body
        .Display   ' Use .Send to fire immediately instead of displaying
    End With

    ' 6. Create follow-up appointment if SLA requires it
    CreateFollowUpAppointment emailType, caseNum, planName

    ' 7. Optional success message — comment out if not wanted
    ' MsgBox "Email created for " & planName & " (Case #" & caseNum & ")", vbInformation

    Set mailItem = Nothing
End Sub

' ----------------------------------------------------------
' CreateFollowUpAppointment
' Creates an Outlook calendar item N days from today.
' Set followUpDays = 0 to skip appointment creation.
'
' Add / adjust cases here to control per-email SLA timing.
' ----------------------------------------------------------
Public Sub CreateFollowUpAppointment(ByVal emailType As String, _
                                       caseNum As String, _
                                       planName As String)

    Dim followUpDays As Long

    Select Case emailType
        Case "Payroll Setup — Final Import"     : followUpDays = 10
        Case "Balance Import Confirmation"       : followUpDays = 3
        Case "Re-Registration Notice"           : followUpDays = 7
        Case "Loan Setup — Takeover Confirmation": followUpDays = 5
        Case "Client Follow-Up — No Response"    : followUpDays = 5
        Case "Onboarding — Welcome Email"        : followUpDays = 0 ' No follow-up
        Case Else                                : followUpDays = 0
    End Select

    If followUpDays = 0 Then Exit Sub

    Dim appt As Outlook.AppointmentItem
    Set appt = Application.CreateItem(olAppointmentItem)

    With appt
        .Subject  = "Follow Up — " & planName & " — " & emailType
        .Body     = "Case #: " & caseNum & vbCrLf & "Plan: " & planName & vbCrLf & _
                    "Email type sent: " & emailType & vbCrLf & _
                    "Follow-up SLA: " & followUpDays & " days"
        .Start    = Date + followUpDays
        .Duration = 30
        .ReminderSet = True
        .ReminderMinutesBeforeStart = 60
        .Save
    End With

    Set appt = Nothing
End Sub

' EmailGeneratorMain
' Entry point — wire your Quick Access Toolbar button here.

End Sub

' ----------------------------------------------------------
' ReadTextFile (helper)
' Reads a .txt template file and returns its contents.
' ----------------------------------------------------------
Private Function ReadTextFile(ByVal filePath As String) As String
    Dim fileNum As Integer
    Dim content As String, line As String

    If Not Dir(filePath) <> "" Then
        MsgBox "Template not found: " & filePath, vbCritical
        ReadTextFile = ""
        Exit Function
    End If

    fileNum = FreeFile
    Open filePath For Input As #fileNum
    Do While Not EOF(fileNum)
        Line Input #fileNum, line
        content = content & line & vbCrLf
    Loop
    Close #fileNum

    ReadTextFile = content
End Function

Option Explicit

' ----------------------------------------------------------
' Form_Initialize
' Fires when form loads — populate TM, TC, and email lists
' ----------------------------------------------------------
Private Sub UserForm_Initialize()
    ' Populate TM combo box
    Dim tmList As Variant, tcList As Variant
    Dim i As Long

    tmList = GetTMList()
    For i = 0 To UBound(tmList, 1)
        cboTM.AddItem tmList(i, 0)
    Next i

    ' Populate TC combo box
    tcList = GetTCList()
    For i = 0 To UBound(tcList, 1)
        cboTC.AddItem tcList(i, 0)
    Next i

    ' Populate email type list
    ' Add / remove entries here to match your template library
    With lstEmailType
        .AddItem "Payroll Setup — Final Import"
        .AddItem "Balance Import Confirmation"
        .AddItem "Re-Registration Notice"
        .AddItem "Loan Setup — Takeover Confirmation"
        .AddItem "Client Follow-Up — No Response"
        .AddItem "Onboarding — Welcome Email"
        ' ... add remaining email types here
    End With

    ' Hide amount field by default
    txtAmount.Visible = False
    lblAmountNote.Visible = False
End Sub

' ----------------------------------------------------------
' lstEmailType_Click
' Show/hide amount field based on email type selection
' ----------------------------------------------------------
Private Sub lstEmailType_Click()
    Dim needsAmount() As String
    needsAmount = Split("Balance Import Confirmation,Loan Setup — Takeover Confirmation", ",")

    Dim showAmt As Boolean
    showAmt = False

    Dim i As Long
    For i = 0 To UBound(needsAmount)
        If lstEmailType.Value = needsAmount(i) Then showAmt = True
    Next i

    txtAmount.Visible = showAmt
    lblAmountNote.Visible = showAmt
End Sub

' ----------------------------------------------------------
' btnSubmit_Click
' Validate inputs then call the email builder
' ----------------------------------------------------------
Private Sub btnSubmit_Click()
    ' Basic validation
    If Trim(txtCaseNum.Value) = "" Or Trim(txtPlanName.Value) = "" Then
        MsgBox "Case number and plan name are required.", vbExclamation
        Exit Sub
    End If
    If cboTM.ListIndex = -1 Or cboTC.ListIndex = -1 Then
        MsgBox "Please select a TM and TC.", vbExclamation
        Exit Sub
    End If
    If lstEmailType.ListIndex = -1 Then
        MsgBox "Please select an email type.", vbExclamation
        Exit Sub
    End If

    ' Resolve TM and TC name + email from the arrays
    Dim tmList As Variant, tcList As Variant
    Dim tmIdx As Long, tcIdx As Long
    tmList = GetTMList() : tmIdx = cboTM.ListIndex
    tcList = GetTCList() : tcIdx = cboTC.ListIndex

    ' Fire the builder
    BuildAndSendEmail _
        emailType := lstEmailType.Value, _
        caseNum   := Trim(txtCaseNum.Value), _
        planName  := Trim(txtPlanName.Value), _
        tmName    := tmList(tmIdx, 0), tmEmail := tmList(tmIdx, 1), _
        tcName    := tcList(tcIdx, 0), tcEmail := tcList(tcIdx, 1), _
        amount    := Trim(txtAmount.Value)

    Me.Hide
End Sub

Private Sub btnCancel_Click()
    Me.Hide
End Sub
SETUP GUIDE
How to install and configure this macro from scratch
INSTALLATION
01
Open the VBE in Outlook
Alt + F11 inside Outlook opens the Visual Basic Editor. You're working in ThisOutlookSession or a new module.
02
Create the four modules
Insert → Module four times. Rename each one in the Properties panel: mod_PeopleLists mod_Templates mod_TokenEngine mod_EmailBuilder
03
Create the UserForm
Insert → UserForm. Rename it frmEmailGenerator. Add the controls listed at the top of the form code module (textboxes, combos, listbox, buttons).
04
Paste each module's code
Copy each module block above into its corresponding VBE module. Paste the form code-behind into the UserForm's code window.
05
Set the template folder path
Update the TEMPLATE_PATH constant in mod_EmailBuilder to point to the folder where your .txt template files live.
TEMPLATE FORMAT
01
Save templates as .txt files
Each email type needs one .txt file in the template folder. The filename must match exactly what you set in mod_Templates.
02
Use curly-bracket tokens
Write your template normally, placing {case} {plan} {TM} {TC} {amount} wherever those values should appear. The engine swaps them at runtime.
03
Use {date+N} for relative dates
For any date that should be relative to today, use {date+0} for today, {date+10} for 10 days out, etc. The N is simply added to today's date.
04
Wire the launch button
In Outlook's Quick Access Toolbar (or any ribbon), add a macro button pointed at EmailGeneratorMain in mod_EmailBuilder. That's the only entry point.
05
Set follow-up SLA days
In CreateFollowUpAppointment(), each email type has a day count. Set to 0 for no appointment. Change the number to adjust the SLA window.
