OVERVIEW
What this tool does, what you can customize, and how it's structured
What It Does
One button → complete email + calendar item
Fill in a case number, plan name, and select TM/TC and email type. The macro loads the matching template from disk, resolves all tokens, opens the drafted email in Outlook, and creates a follow-up appointment in your calendar — all in one click.
What You Can Customize
All data is isolated from the engine
  • TM and TC rosters — edit one function in mod_PeopleLists
  • Email types and template filenames — one Select Case in mod_Templates
  • Template body — edit the .txt files directly, no VBA needed
  • Token values — add new {tokens} with a single Replace() call
  • Follow-up SLA days — one Select Case in mod_EmailBuilder
  • Send vs Display — one property change per email type
Structure at a Glance
5 components, clean separation of concerns
  • frmEmailGenerator — UserForm: inputs + submit
  • mod_PeopleLists — TM/TC name + email arrays
  • mod_Templates — email type → template file mapping
  • mod_TokenEngine — {token} resolver including {date+N}
  • mod_EmailBuilder — assembly, dispatch, appointment creator

' ============================================================
' mod_Templates
' GetTemplateAndMessage() — given an email type, returns:
'   (1) the template filename to load from disk
'   (2) the custom reminder message shown before sending
'
' To add a new email type:
'   1. Add a Case block here with template name + message
'   2. Save the matching .txt file in your templates folder
'   3. Add the type name to lstEmailType in the UserForm Initialize
' ============================================================

Option Explicit

Public Sub GetTemplateAndMessage(ByVal emailType As String, _
                                  templateFile As String, _
                                  customMsg    As String)

    Select Case emailType

        Case "Payroll Setup — Final Import"
            templateFile = "tmpl_payroll_final_import.txt"
            customMsg    = "Reminder: Attach the completed EDS Layout file before sending."

        Case "Balance Import Confirmation"
            templateFile = "tmpl_balance_import_confirm.txt"
            customMsg    = "Reminder: Confirm wire has been booked in cashiering before sending."

        Case "Re-Registration Notice"
            templateFile = "tmpl_rereg_notice.txt"
            customMsg    = "Reminder: Attach the completed Re-Reg form. Submit at least 4 weeks before liquidation."

        Case "Loan Setup — Takeover Confirmation"
            templateFile = "tmpl_loan_takeover.txt"
            customMsg    = "Reminder: Verify Loan Header balances match Loan Source totals before sending."

        Case "Client Follow-Up — No Response"
            templateFile = "tmpl_client_followup.txt"
            customMsg    = "No attachments needed. Check case notes before sending."

        Case "Onboarding — Welcome Email"
            templateFile = "tmpl_onboarding_welcome.txt"
            customMsg    = "Reminder: CC the assigned TM and include the welcome packet PDF."

        ' Add additional email types here following the same pattern

        Case Else
            templateFile = ""
            customMsg    = "No custom message configured for this email type."

    End Select

End Sub

mod_TokenEngine.bas 
' ============================================================
' mod_TokenEngine
' ReplaceTokens() — takes the raw template string and swaps
' every {token} with its live value.
'
' Static tokens:  {case}  {plan}  {TM}  {TC}  {amount}
' Dynamic tokens: {date+N} where N = days to add to today
'   {date+0}  → today
'   {date+2}  → 2 days from today
'   {date+10} → 10 days from today
'
' To add a new static token: add one Replace() call below.
' ============================================================

Option Explicit

Public Function ReplaceTokens(ByVal body    As String, _
                               caseNum  As String, _
                               planName As String, _
                               tmName   As String, _
                               tcName   As String, _
                               amount   As String) As String

    ' ── Static token replacement ──────────────────────────────
    body = Replace(body, "{case}",   caseNum)
    body = Replace(body, "{plan}",   planName)
    body = Replace(body, "{TM}",    tmName)
    body = Replace(body, "{TC}",    tcName)
    body = Replace(body, "{amount}", amount)

    ' ── Dynamic date token replacement ────────────────────────
    ' Scans for any {date+N} pattern and resolves to a date string.
    ' Example: {date+10} → "03/20/2026" when today is 03/10/2026
    Dim tokenStart As Long, tokenEnd As Long
    Dim token As String, daysStr As String
    Dim daysToAdd As Long

    Do
        tokenStart = InStr(body, "{date+")
        If tokenStart = 0 Then Exit Do

        tokenEnd = InStr(tokenStart, body, "}")
        If tokenEnd = 0 Then Exit Do

        token   = Mid(body, tokenStart, tokenEnd - tokenStart + 1)
        daysStr = Mid(token, 7, Len(token) - 7)
        If Not IsNumeric(daysStr) Then Exit Do

        daysToAdd = CLng(daysStr)
        body = Replace(body, token, Format(Date + daysToAdd, "mm/dd/yyyy"))
    Loop

    ReplaceTokens = body

End Function

mod_EmailBuilder.bas 
' ============================================================
' mod_EmailBuilder
' Three subs + one helper:
'   BuildAndSendEmail()         — main assembly + dispatch
'   CreateFollowUpAppointment() — SLA-based calendar entry
'   EmailGeneratorMain()        — entry point (wire button here)
'   ReadTextFile()              — loads .txt template from disk
' ============================================================

Option Explicit

' Update this path to point at your templates folder
Private Const TEMPLATE_PATH As String = "C:\Templates\EmailGen\"

' ----------------------------------------------------------
' BuildAndSendEmail — called by btnSubmit_Click
' ----------------------------------------------------------
Public Sub BuildAndSendEmail(ByVal emailType As String, _
                              caseNum  As String, _
                              planName As String, _
                              tmName   As String, tmEmail As String, _
                              tcName   As String, tcEmail As String, _
                              amount   As String)

    Dim templateFile As String, customMsg As String
    Dim body         As String
    Dim mailItem     As Outlook.MailItem

    ' 1. Get template filename + pre-send reminder
    GetTemplateAndMessage emailType, templateFile, customMsg

    If templateFile = "" Then
        MsgBox "No template configured for: " & emailType, vbExclamation
        Exit Sub
    End If

    ' 2. Load template from disk
    body = ReadTextFile(TEMPLATE_PATH & templateFile)

    ' 3. Replace all tokens (static + dynamic dates)
    body = ReplaceTokens(body, caseNum, planName, tmName, tcName, amount)

    ' 4. Show pre-send reminder
    If customMsg <> "" Then
        MsgBox customMsg, vbInformation, "Before You Send"
    End If

    ' 5. Build and display the mail item
    Set mailItem = Application.CreateItem(olMailItem)
    With mailItem
        .To      = tmEmail & ";" & tcEmail
        .Subject = emailType & " — Case #" & caseNum & " — " & planName
        .Body    = body
        .Display   ' Change to .Send to fire immediately
    End With

    ' 6. Create follow-up appointment if SLA requires it
    CreateFollowUpAppointment emailType, caseNum, planName

    ' Optional: uncomment the line below for a success popup
    ' MsgBox "Email created for " & planName & " (Case #" & caseNum & ")", vbInformation

    Set mailItem = Nothing
End Sub

' ----------------------------------------------------------
' CreateFollowUpAppointment
' SLA day counts per email type. Set to 0 for no appointment.
' ----------------------------------------------------------
Public Sub CreateFollowUpAppointment(ByVal emailType As String, _
                                       caseNum  As String, _
                                       planName As String)

    Dim followUpDays As Long

    Select Case emailType
        Case "Payroll Setup — Final Import"      : followUpDays = 10
        Case "Balance Import Confirmation"        : followUpDays = 3
        Case "Re-Registration Notice"            : followUpDays = 7
        Case "Loan Setup — Takeover Confirmation" : followUpDays = 5
        Case "Client Follow-Up — No Response"     : followUpDays = 5
        Case "Onboarding — Welcome Email"         : followUpDays = 0
        Case Else                                 : followUpDays = 0
    End Select

    If followUpDays = 0 Then Exit Sub

    Dim appt As Outlook.AppointmentItem
    Set appt = Application.CreateItem(olAppointmentItem)

    With appt
        .Subject  = "Follow Up — " & planName & " — " & emailType
        .Body     = "Case #: " & caseNum & vbCrLf & "Plan: " & planName & vbCrLf & _
                    "Email sent: " & emailType & vbCrLf & _
                    "Follow-up SLA: " & followUpDays & " days"
        .Start    = Date + followUpDays
        .Duration = 30
        .ReminderSet = True
        .ReminderMinutesBeforeStart = 60
        .Save
    End With

    Set appt = Nothing
End Sub

' EmailGeneratorMain — entry point, wire your button here

' ----------------------------------------------------------
' ReadTextFile — loads a .txt file and returns its contents
' ----------------------------------------------------------
Private Function ReadTextFile(ByVal filePath As String) As String
    Dim fileNum  As Integer
    Dim content  As String
    Dim line     As String

    If Not Dir(filePath) <> "" Then
        MsgBox "Template not found: " & filePath, vbCritical
        ReadTextFile = ""
        Exit Function
    End If

    fileNum = FreeFile
    Open filePath For Input As #fileNum
    Do While Not EOF(fileNum)
        Line Input #fileNum, line
        content = content & line & vbCrLf
    Loop
    Close #fileNum

    ReadTextFile = content
End Function

' ============================================================
' frmEmailGenerator — UserForm code-behind
' Paste into the form's code window in VBE.
'
' Controls needed (create in the form designer):
'   txtCaseNum    — TextBox
'   txtPlanName   — TextBox
'   txtAmount     — TextBox (hidden by default)
'   cboTM         — ComboBox
'   cboTC         — ComboBox
'   lstEmailType  — ListBox
'   btnSubmit     — CommandButton
'   btnCancel     — CommandButton
'   lblAmountNote — Label (shows/hides with amount field)
' ============================================================

Option Explicit

Private Sub UserForm_Initialize()
    Dim tmList As Variant, tcList As Variant
    Dim i      As Long

    ' Populate TM combo box
    tmList = GetTMList()
    For i = 0 To UBound(tmList, 1)
        cboTM.AddItem tmList(i, 0)
    Next i

    ' Populate TC combo box
    tcList = GetTCList()
    For i = 0 To UBound(tcList, 1)
        cboTC.AddItem tcList(i, 0)
    Next i

    ' Populate email type list — keep in sync with mod_Templates
    With lstEmailType
        .AddItem "Payroll Setup — Final Import"
        .AddItem "Balance Import Confirmation"
        .AddItem "Re-Registration Notice"
        .AddItem "Loan Setup — Takeover Confirmation"
        .AddItem "Client Follow-Up — No Response"
        .AddItem "Onboarding — Welcome Email"
    End With

    ' Hide amount field by default — shown only for relevant email types
    txtAmount.Visible     = False
    lblAmountNote.Visible = False
End Sub

Private Sub lstEmailType_Click()
    ' Show amount field only for email types that need it
    Dim needsAmount() As String
    needsAmount = Split("Balance Import Confirmation,Loan Setup — Takeover Confirmation", ",")

    Dim showAmt As Boolean
    showAmt = False

    Dim i As Long
    For i = 0 To UBound(needsAmount)
        If lstEmailType.Value = needsAmount(i) Then showAmt = True
    Next i

    txtAmount.Visible     = showAmt
    lblAmountNote.Visible = showAmt
End Sub

Private Sub btnSubmit_Click()
    ' Validate required fields
    If Trim(txtCaseNum.Value) = "" Or Trim(txtPlanName.Value) = "" Then
        MsgBox "Case number and plan name are required.", vbExclamation
        Exit Sub
    End If
    If cboTM.ListIndex = -1 Or cboTC.ListIndex = -1 Then
        MsgBox "Please select a TM and TC.", vbExclamation
        Exit Sub
    End If
    If lstEmailType.ListIndex = -1 Then
        MsgBox "Please select an email type.", vbExclamation
        Exit Sub
    End If

    ' Resolve TM and TC from the people list arrays
    Dim tmList As Variant, tcList As Variant
    Dim tmIdx  As Long,   tcIdx  As Long
    tmList = GetTMList() : tmIdx = cboTM.ListIndex
    tcList = GetTCList() : tcIdx = cboTC.ListIndex

    BuildAndSendEmail _
        emailType := lstEmailType.Value,        _
        caseNum   := Trim(txtCaseNum.Value),  _
        planName  := Trim(txtPlanName.Value),  _
        tmName    := tmList(tmIdx, 0), tmEmail := tmList(tmIdx, 1), _
        tcName    := tcList(tcIdx, 0), tcEmail := tcList(tcIdx, 1), _
        amount    := Trim(txtAmount.Value)

    Me.Hide
End Sub

Private Sub btnCancel_Click()
    Me.Hide
End Sub
SETUP GUIDE
How to install and configure this macro from scratch
INSTALLATION
01
Open the VBE in Outlook
Alt + F11 inside Outlook opens the Visual Basic Editor. You're working inside the Outlook project, not Excel.
02
Create the four modules
Insert → Module four times. Rename each in the Properties panel: mod_PeopleLists mod_Templates mod_TokenEngine mod_EmailBuilder
03
Create the UserForm
Insert → UserForm. Rename it frmEmailGenerator. Add all controls listed at the top of Module 05 above.
04

Copy each block above into its VBE module. Paste the form code into the UserForm's code window (not a standard module).
05

Update the TEMPLATE_PATH constant at the top of mod_EmailBuilder to point at the folder where your .txt files live.
06
Wire the launch button
In Outlook's Quick Access Toolbar, add a macro button pointed at EmailGeneratorMain in mod_EmailBuilder. That's the only entry point.
TEMPLATE FORMAT
01
Save templates as .txt files
Each email type needs one .txt file in the template folder. The filename must exactly match what you set in mod_Templates.
02
Use curly-bracket tokens for dynamic values
Write your template normally. Place {case} {plan} {TM} {TC} {amount} wherever those values should appear. The engine replaces them at runtime.
03
Use {date+N} for relative dates
For dates relative to today: {date+0} = today, {date+10} = 10 days out. Use as many as needed in a single template — all resolved in one pass.
04
Add new email types easily
Add a Case block in mod_Templates, save the .txt file, and add the type name to lstEmailType in the UserForm Initialize. No other changes needed.
05
Configure follow-up SLA days
In CreateFollowUpAppointment(), set the day count per email type. Set to 0 for no appointment. Adjust numbers to match your SLA windows.
06
Send vs Display
In BuildAndSendEmail(), the mail item currently uses .Display for review. Change to .Send to fire immediately — can be toggled per email type if needed.
