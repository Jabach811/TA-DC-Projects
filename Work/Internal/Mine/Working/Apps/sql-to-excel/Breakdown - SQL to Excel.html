OVERVIEW
What this workbook does, the problem it solves, and how it's structured
The Problem It Solves
No more manual query editing between plans
Every time you move to a new plan you'd have to open AQT, find the query, highlight the case number, type the new one in the right format, hope there's no trailing space, and re-run. This workbook eliminates all of that. Enter the case number once, every query updates and refreshes automatically.
What It Does
One entry point, all tables refreshed
  • UserForm accepts raw case number input
  • Validator detects prefix type (Q/N vs T/J) and enforces spacing rules
  • Builds all required query format variants automatically
  • Injects formatted values into each Power Query connection string
  • Refreshes all tables — data lands pre-formatted in Excel
  • Navigate tab by tab — each sheet is its own extraction table

Key design: The format normalization layer is fully decoupled from the query injection layer. Adding a new query tab only requires adding one connection and one format variant — the validation and refresh logic stays unchanged.
Format rules: Q/N prefix → 2 letters + 5 digits + 3 spaces + 5 digits  |  T/J prefix → 2 letters + 6 digits + 2 spaces + 5 digits. Wrong spacing = broken query — the validator enforces this before any injection occurs.

' ============================================================
' mod_CaseValidator
'
' Plan number format rules:
'   Q / N prefix → 2 letters + 5 digits + 3 spaces + 5 digits
'                  e.g. "QK12345   67890"
'   T / J prefix → 2 letters + 6 digits + 2 spaces + 5 digits
'                  e.g. "TA123456  67890"
'
' BuildFormatVariants returns a Collection of named format strings
' for use by mod_QueryBuilder when injecting into connections.
' ============================================================

Option Explicit

Public Function DetectPrefixType(ByVal caseNum As String) As String
    Dim prefix As String
    prefix = UCase(Left(caseNum, 1))

    Select Case prefix
        Case "Q", "N" : DetectPrefixType = "QN"
        Case "T", "J" : DetectPrefixType = "TJ"
        Case Else       : DetectPrefixType = "UNKNOWN"
    End Select
End Function
